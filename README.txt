APÓS EXECUTAR O INSTALADOR DO APIHUB SIGA OS PASSOS ABAIXO PARA A CONFIGURAÇÃO COMPLETA DO APIHUB E REDIS:
-------------------------------------------------------------------------------------------------------------------
1 - CONFIGURAÇÃO DO ENV:
Acesse ao App Gerenciador Apihub e clique no Botão ENV para configurar os dados conforme solicitado na tela e em seguida clique em OK para salvar.

------------------------------------------------------------------------------------------------------------------------

2 - Após a configuração do arquivo ENV, dentro do App Gerenciador Apihub clique no botão PLAY ▶️ para instalar os serviços e iniciá-los.
Com isso, acesse ao dashboard através do atalho na área de trabalho ou acessando no navegador http://127.0.0.1\dashboard para verificar se a aplicação está sendo executada.
é possível verificar direto no Gerenciador Apihub se o serviço está ativo, mas para melhor detalhes abra o dashboard ou então clique no botão Logs

Caso não abra o dashboard do Apihub, acesse em C:\INFARMA\APIHUB\logs\ para verificar os erros e mensagens apresentados ou solicite apoio do suporte N2.
